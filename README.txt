== Description ==

This plugin works only with [RealHomes WordPress Real Estate Theme](https://realhomes.io/).

Provides Facebook, Google and Twitter social login support for RealHomes theme.

Plugin includes translation files for Spanish, Italian, French, German, Turkish and Portuguese languages.

== Installation ==

This section describes how to install the plugin and get it working.

1. You must have RealHomes theme properly installed on your website with required plugins.
2. Upload `realhomes-social-login` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to the *Dashboard > Easy Real Estate > Social Login Settings* page for the configuration.
